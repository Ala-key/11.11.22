How to create local repository

  1. Right click IT folder using the mouse
  2. Select "Git Bash Here"
  3. Execute "mklir folder name"
  4. Ececute cd Folder name
  5. Execute git init

    6. How to link local repository to GitHub

    1. Go to new repository
    2. Fill repository name field and press "Create repository"
    3. Execute "git remote add origin"
